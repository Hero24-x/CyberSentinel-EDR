# CyberSentinel EDR

Lightweight Endpoint Detection and Response platform built using Python and Flask.

## Features

- Process Monitoring
- CPU Monitoring
- Network Monitoring
- File Integrity Monitoring
- Alert Management
- Dashboard
- Report Generation

## Installation

```bash
git clone https://github.com/username/CyberSentinel-EDR
cd CyberSentinel-EDR

pip install -r requirements.txt
```

Run:

```bash
python run.py
```

Open:

http://127.0.0.1:5000
